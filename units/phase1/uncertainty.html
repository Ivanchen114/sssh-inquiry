📏 有效數字與測量不確定度

從端點還是中間點按碼錶——這個問題，你需要不確定度才能回答

論證建模 W2 · 2 節課 A1 整理數據 A2 分析詮釋
📌 教學目標（教師視角）
  • 用「爭論 → 實驗 → 數據 → 不確定度」驅動知識需求，讓概念從情境中自然浮現
  • 建立 A 類（統計）與 B 類（儀器）不確定度的直覺區別
  • 讓學生第一次體驗「數據有差，不代表結論有差」的科學判斷

  • 能用儀器最小刻度計算 B 類不確定度
  • 能合成 u_c 並以「x̄ ± U」格式正確報告測量結果
  • 能用不確定度區間判斷兩組數據是否真的不同

教師提示

大多數班級：兩組區間會重疊，說明人的反應時間（A類）才是主要誤差來源，不是選哪個點。這個「出乎意料」的結論本身就是很好的教學時刻——物理直覺和數據不符，要相信數據。

🔵 中間派

1-1 有效數字
觀念 有效數字的判讀

測量值中每一個有意義的數字都算有效數字。

規則範例有效位數
非零數字一律算14.33 位
兩非零之間的零算1.0234 位
末位小數的零算1.203 位
小數點前的零不算0.00892 位
整數末位的零，用科學記號澄清1.30 × 10³3 位
觀念 有效數字的運算
加法 / 減法
結果的小數位數取決於最少小數位數的那個數。
12.35 + 1.4 → 13.8
（1.4 只到小數第 1 位）
乘法 / 除法
結果的有效位數取決於最少有效位數的那個數。
3.14 × 2.5 → 7.9
（2.5 只有 2 位有效數字）
練習 A · 有效數字

判讀題：說出下列各數的有效數字位數。

① 0.00520
② 1.030 × 10⁴
③ 300（無額外資訊）
④ 2.400

計算題：

⑤ 計算 3.76 + 0.2 + 14.135，並保留正確有效數字。
⑥ 計算 6.24 × 0.53，並保留正確有效數字。
1-2 測量不確定度
觀念 不確定度的意義

任何測量都不完美。不確定度描述「真實值可能落在哪個範圍內」——數值越小，測量越可靠。完整的測量結果寫成：

測量結果 = x̄ ± U

x̄ 為平均值（最佳估計值）；U 為擴充不確定度（真值有約 95% 的機率落在此範圍內）

A 類不確定度 u_A
來自多次重複測量的統計分析，反映測量結果的隨機分散程度。
B 類不確定度 u_B
來自儀器特性或已知規格，不需重複測量，由解析度直接推算。
觀念 A 類不確定度：用統計計算
  1. 計算平均值（最佳估計值）
    x̄ = (x₁ + x₂ + ⋯ + xₙ) / n
  2. 計算樣本標準差 s（反映每次測量的分散程度）
    s = √[ Σ(xᵢ − x̄)² / (n − 1) ]
  3. 計算 A 類不確定度（平均值的標準誤差）
    u_A = s / √n
    → 測量次數 n 越多，u_A 越小（平均值越可靠）
例題 A 類：碼錶測單擺週期（端點計時，5 次）

某組同學從端點計時的 5 次紀錄（10 次擺動總時間 ÷ 10，單位：s）：

次數T₁T₂T₃T₄T₅
週期 (s)0.890.910.900.920.89
x̄ = (0.89 + 0.91 + 0.90 + 0.92 + 0.89) / 5 = 0.902 s
s = √[((0.89−0.902)²+(0.91−0.902)²+(0.90−0.902)²+(0.92−0.902)²+(0.89−0.902)²) / 4]
≈ 0.013 s
u_A = 0.013 / √5 ≈ 0.0058 s
例題 A 類：量繩長 6 次（含離群值討論）

6 次測量結果（cm）：20.1、20.2、20.0、20.1、20.5、20.1

⚠️ 20.5 和其他值差了約 0.4 cm。討論：這是正常的隨機分散，還是操作失誤？保留 vs 剔除對結果有多大影響？
x̄ = (20.1+20.2+20.0+20.1+20.5+20.1) / 6 ≈ 20.17 cm
s ≈ 0.173 cm（保留 20.5 的情況）
u_A = 0.173 / √6 ≈ 0.071 cm
觀念 B 類不確定度：從儀器解析度推算

若真值在最小刻度範圍內均勻分布（矩形分布），B 類不確定度為：

u_B = 最小刻度 / (2√3)
指針式 / 刻度盤：
最小刻度 = 最小格值
例：0.1 cm 尺 → u_B = 0.1/(2√3) ≈ 0.029 cm
數位顯示儀器：
最小刻度 = 末位數字 1 個單位
例：顯示到 0.01 s → u_B = 0.01/(2√3) ≈ 0.0029 s
例題 B 類：數位碼錶（最小刻度 0.01 s）
最小刻度 = 0.01 s
u_B = 0.01 / (2 × 1.732) ≈ 0.0029 s

注意：u_B ≈ 0.003 s 遠小於 u_A ≈ 0.006 s——碼錶本身的解析度不是主要誤差來源，人的反應時間才是。

觀念 組合不確定度與報告格式

A 類與 B 類同時存在時，以平方和開根號合成：

u_c = √(u_A² + u_B²)

再乘擴充因子 k = 2（對應約 95% 信心水準）：

U = 2 × u_c
📋 數據報告規範（GUM 準則）
  1. U 通常保留 2 位有效數字，且採無條件進位（後面有數字就往上加）
  2. x̄ 的末位數必須對齊 U 的末位數
✓ 正確：14.160 ± 0.050 cm
✗ 錯誤：14.3 ± 0.02 cm（位數沒對齊）
例題 完整計算：單擺端點計時組的最終結果
// 承接前面計算：
u_A = 0.0058 s
u_B = 0.01 / (2√3) ≈ 0.0029 s
u_c = √(0.0058² + 0.0029²) ≈ 0.0065 s
U = 2 × 0.0065 = 0.013 → 無條件進位 → 0.014 s
x̄ = 0.902 s → 對齊末位 → 0.902 s
最終結果：T_端點 = 0.902 ± 0.014 s
回答開場的問題
⚖️ 用你的數據驗證：端點 vs 中間點，差異有意義嗎？

分別計算兩組的完整結果（x̄ ± U），然後比較兩個區間：

端點計時
T₁ = x̄₁ ± U₁
區間：[ x̄₁ − U₁ , x̄₁ + U₁ ]
中間點計時
T₂ = x̄₂ ± U₂
區間：[ x̄₂ − U₂ , x̄₂ + U₂ ]
判斷標準：
✅ 若兩個區間重疊 → 差異不顯著，無法判定哪個方法更準
❌ 若兩個區間不重疊 → 兩種方法有系統性差異
💡 預期結論：大多數情況下兩組區間會重疊。這說明： 測量的主要不確定度來自人的反應時間（u_A 較大），而不是選端點或中間的差異。 選哪個點不是關鍵，每次保持一致才是。
課堂討論引導
  • 「你原來的假設是什麼？數據支持你了嗎？如果沒有，你願意修正假設嗎？」
  • 「如果我們測 20 次而不是 5 次，u_A 會怎麼變？為什麼？」
  • 「什麼樣的操作方式可以讓 u_A 更小？（提示：更一致的按錶時機）」
練習 B · 不確定度計算
① B 類：一支體重計，最小刻度 0.5 kg。計算 u_B。
② A 類：5 次測量彈簧伸長量（cm）：3.5、3.7、3.6、3.8、3.5。
計算 x̄、s、u_A，並以 x̄ ± U 格式報告結果。
③ 完整流程：用最小刻度 1 mm 的直尺量長度，4 次數據（mm）：152、154、153、153。
計算 u_A、u_B、u_c、U，並寫出完整結果。
📎 補充：不確定度的傳播（本單元不要求，W3 單擺實驗時會用到）

如果最終結果 Z 是由多個測量量 A、B 計算而來，不確定度會一起「傳播」過去：

加法 / 減法　Z = A ± B
u_Z = u_A + u_B
絕對不確定度直接相加
乘法 / 除法　Z = A × B 或 A/B
u_Z/Z = u_A/A + u_B/B
相對不確定度相加

→ 下一單元（W3）計算 g = 4π²L/T² 時，L 和 T 各自有不確定度，就需要用乘除的傳播規則。

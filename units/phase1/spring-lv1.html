🌀 彈簧 · Lv1 論證建模

虎克定律虛擬實驗室——彈力 F 和伸長量 x 是什麼關係？你來發現。

論證建模 3 節課 線性關係 + 擬合
🎯 學習目標
  • 能從數據中辨識趨勢與規律（例如：從 F vs x 圖判斷線性關係）
  • 能用數學模型（圖表、公式）描述觀察到的關係（例如：從 F vs x 圖中發現線性關係並求斜率）

  • 能評估模型的適用範圍與限制
階段一 · 發現問題
在動手之前，先用直覺預測「力與伸長量」的關係

Lv1 老師已經把研究問題定好了：F 和 x 是什麼數學關係？你的工作是——在操作虛擬彈簧前，把你的直覺預測寫下來。做完實驗回來對照，你會親眼看到「直覺 vs 數據」的落差。

🔒 研究問題：老師提供 ✏️ 直覺預測：你來寫
在操作虛擬彈簧之前，先用直覺回答三個預測題。
P1：如果掛的重量加倍（100 gw → 200 gw），彈簧伸長量也會剛好加倍嗎？
P2：「力」和「伸長量」的圖，你猜會是直線還是曲線？如果是直線，經過原點嗎？
P3：如果一直加重量，彈簧會一直等比例伸長嗎？有沒有極限？
✏️ 把你的三個預測寫下來（P1–P3），實驗完回來對照。
🎯 根據直覺，當重量加倍時，伸長量會怎樣？
很多人的直覺是線性關係——力加倍、伸長也加倍。但這個直覺對不對？等下用數據來驗證！有趣的是：即使很多情況下直覺是對的，極端條件下往往會失效。
階段二 · 規劃研究
看老師怎麼設計這個研究：實驗裝置 + 變因 + 任務

Lv1 你不用自己規劃實驗——老師已經把要測什麼（F 和 x）、怎麼操作（掛砝碼、量伸長量）、重複次數（每種 3 次）都定好了。下一 Lv 你就要自己設計實驗了。

彈簧的基本行為
彈簧受力時會伸長——掛越重的砝碼，伸得越多。但「多多少」？是等比例增加，還是越來越難拉開？
今天的任務
力 F 和伸長量 x 之間是什麼數學關係？你能從數據中找出來嗎？
💡 你的任務：選擇不同重量的砝碼掛在彈簧下，記錄每次的伸長量（至少 5 種重量，每種記錄 3 次）。完成後，畫出 F vs x 圖——觀察它的形狀，找出力和伸長量之間的數學關係。
彈簧原長 $L_0 = 8.0$ cm 伸長量 $x = $ 目前長度 $- L_0$ $F = $ 砝碼重量（gw）$\times g$
🤔 想一想：如果 F 和 x 是正比關係，圖形應該長什麼樣？

正比關係的圖是一條過原點的直線。因為當力 $F = 0$ 時，彈簧沒有伸長 $x = 0$。如果你畫的圖不過原點，可能是原長測量有誤。做完實驗後，回來看你的圖是不是真的是直線——也許你會發現更複雜的關係！

💭 模型選擇與論證：
Q1：選線性趨勢線。截距代表「力 = 0 時的伸長量」——物理上沒掛砝碼時，彈簧伸長量應該是多少？你的截距合理嗎？

Q4：用「主張—理由—證據」論證：力和彈簧伸長量之間是什麼關係？你選擇哪條趨勢線，為什麼？
✏️ 用「主張—理由—證據」論證你選的模型
🎯 哪個說法最準確描述 F 和 x 的關係？
從你的數據來看，F 和 x 確實成正比——通過原點的直線。斜率的單位是 N/m，代表「讓彈簧多伸長 1 m 需要多少 N 的力」，就是彈簧的硬度指標。不過 A 也有道理：真實彈簧在極端拉伸下會偏離線性——這是 Lv2 可以探究的問題。
🤔 想一想：為什麼同一個重量測 3 次，伸長量 x 每次都略有不同？

虛擬實驗特意加入了噪聲（±0.3 cm），模擬真實測量的分散：彈簧不是完美的、砝碼位置略有偏差、摩擦不同…這就是「測量不確定度」。你的三次數據越接近，代表實驗重複性越好。

💡 原來前人也發現過——虎克定律

你從數據中發現 F 和 x 呈直線關係。17 世紀的物理學家虎克（Robert Hooke）也發現了同樣的事，後人把它稱為虎克定律：

$F = kx$

其中 $k$ 是彈力常數（又叫彈性係數），單位 N/m，就是你圖上的斜率。$k$ 越大代表彈簧越「硬」。試試看——你的斜率就是你這條彈簧的 $k$ 值，跟 350 年前虎克做的是同一件事。

📈 冪級數與線性化

讓曲線變直——用取 log 把冪次關係轉成直線

規劃研究 W8 · 2 節課 數據分析工具
本單元學習目標
  • 理解冪次關係 y = axⁿ 的圖形長什麼樣子
  • 能用取對數（log-log 圖）把冪次關係線性化
  • 能從 log-log 圖的斜率判斷冪次 n 的值
  • 能選擇適合的圖形化方式表達數據的關係
⏱ 課堂節奏

第一節：冪次關係概念 + 手動計算 log 值。
第二節：Excel 操作（log 轉換 + log-log 圖 + 斜率解讀）。

數學底線說明

不需要學生精熟對數計算，重點是「讓圖變直、從斜率讀冪次」的操作流程。計算機或 Excel 做 log 即可。

情境引入
為什麼曲線很麻煩？
原始圖（曲線）
y = x²
y vs x：曲線，無法算斜率
線性化圖（直線）✓
log y = 2 log x
log y vs log x：直線，斜率 = 冪次
核心概念
線性化的步驟
1
原始關係
y = a · xⁿ
2
兩邊取 log
log y = log a + n · log x
3
令 Y = log y，X = log x
Y = (log a) + n · X　← 這是直線！
✓
結論：log-log 圖的斜率 = 冪次 n
截距 = log a，所以 a = 10^(截距)
常見的冪次關係例子
物理情境關係冪次 nlog-log 斜率
單擺週期 vs 擺長T ∝ L^(1/2)0.50.5
行星公轉週期 vs 軌道半徑T ∝ r^(3/2)1.51.5
自由落體距離 vs 時間d ∝ t²22
彈簧彈力 vs 伸長量F ∝ x¹11（線性）

討論引導問題
  • 「斜率是 1.52，我說這個系統的冪次是 3/2——你同意嗎？誤差在實驗上能接受嗎？」
  • 「如果 log-log 圖的斜率不是整數也不是簡單分數怎麼辦？代表什麼？」
  • 「線性化的技術除了 log-log，還有什麼？（提示：半log圖，用於指數衰減）」
答案

衛星數據的斜率應接近 1.5（克卜勒第三定律：T² ∝ r³，即 T ∝ r^1.5）。這是一個很好的「數據驗證定律」的範例。

延伸思考
  1. 如果兩個變因不是冪次關係，而是指數關係（y = a·bˣ），取什麼 log 會讓它變直線？斜率代表什麼？
  2. 回顧單擺實驗：如果你畫 T vs L（原始），和 T² vs L（線性化），哪個更好用？為什麼？
  3. 科學家如何用 log-log 圖判斷一個關係「是不是」冪次關係？如果不是，圖形會有什麼特徵？

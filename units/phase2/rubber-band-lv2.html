🥫 橡皮筋動力罐模組 · Lv2 規劃研究

Lv1 我們知道轉圈數會影響距離。但影響的因素遠不止這一個——你來自己設計實驗。

規劃研究 4 節課 自主設計
🎯 學習目標
  • 能從現象或理論出發，提出可驗證的研究問題（例如：橡皮筋條數如何影響推出距離？）
  • 能設計含操縱變因、控制變因與應變變因的實驗計畫（例如：控制初始推力、地面材質）
  • 能有系統地蒐集數據，並判斷數據的可靠性（重複測量至少 3 次，檢查標準差）
  • 能用圖表呈現結果，並以數據為基礎進行分析與發表
📖 情境故事

你發現橡皮筋動力罐很有趣，但 Lv1 只測了「轉圈數」這一個變因。其實影響推出距離和回滾距離的因素很多——橡皮筋的條數、粗細、罐子重量、地面材質、重物位置。
如果要做出「推最遠」或「回滾最準」的動力罐，你會怎麼研究？

可能的變因有哪些？
🎀 橡皮筋
  • 條數：1 / 2 / 3 / 4
  • 粗細（寬度）：細 / 中 / 粗
  • 長度：短 / 中 / 長
  • 材質：天然橡皮筋 / 合成 / 髮圈
🔩 罐子與環境
  • 罐子重量：加砝碼到不同克數
  • 罐子直徑：不同型號罐子
  • 重物質量：不同重量的螺帽
  • 地面材質：磁磚 / 木板 / 地毯
  • 初始推力：輕輕推 / 中等推 / 用力推
📝 實驗計畫表
1. 研究問題：「___________是否影響推出距離/回滾距離？」
2. 理論預測：根據 Lv1 的能量觀點，我認為 ___________ ，因為 ___________
3. 操縱變因：___________ （至少 4 個水準）：___________
4. 控制變因：___________ （列出至少 3 個，含控制方式）
5. 應變變因：推出距離 / 回滾距離（測量方式：___________ ）
6. 重複次數：每個水準測 ___________ 次
💡 提示：變因越多，實驗越複雜。建議初期選一個「最想探究」的變因，其他都嚴格控制。

① 可重現性：如果我照著你的步驟做，能得到一樣的結果嗎？哪裡不夠清楚？

③ 數據量：水準夠多嗎？重複次數夠嗎？能看出趨勢嗎？
④ 一個建議：如果你能改一個地方，你會改什麼？
⑤ 研究問題：這組的研究問題夠明確嗎？能用一句話說出他們在探究什麼嗎？

🎯 你選了「橡皮筋條數如何影響推出距離」。這實驗中哪個是操縱變因？
正確！操縱變因是你主動改變的因素（條數），距離是你測量的結果（應變變因）。如果你改條數又改轉圈數，就沒辦法知道「到底是誰影響了距離」。這就是為什麼要「只改一個」。

🤔 想一想：你第三次測量某個水準時，數據明顯異常（遠小於另外兩次）。你該怎麼辦？

不要直接刪掉！先想想可能原因：罐子撞到障礙物？推力不足？地面有障礙？如果你找到明確的「操作失誤」原因，可以標記它，再做一次補測。如果找不到原因，保留它——報告裡說「有一筆數據離群，原因不明」比偷偷刪掉更誠實。誠實面對限制，本身就是科學素養。

Q1：你的結果支持理論預測嗎？用不確定度範圍說明「有顯著差異」或「沒有顯著差異」。
Q2：畫「操縱變因 vs 應變變因」的散佈圖（含誤差棒）。趨勢是直線、曲線還是無規律？
Q3：全班各組的結果合在一起，有互相矛盾的嗎？怎麼解釋（可能是控制變因不同、測量誤差、或真的有其他隱藏變因）？
🎯 你的數據顯示：1 條橡皮筋推出 2.1±0.2 m，2 條推出 4.0±0.3 m。下列哪個結論最合適？
正確！這是判定「有顯著差異」的科學方法：不確定度範圍不重疊 → 變因確實有影響。A 項誤會了：有誤差不代表不可信，反而要用誤差來判定。C 項用直覺而非數據，不科學。

不確定度檢驗
每個水準至少量 3 次，算出 uA = s/√n
兩組數據的不確定度範圍有沒有重疊？
有重疊 → 「沒有顯著差異」
無重疊 → 「有顯著差異」
圖表分析
用 Excel 或手畫散佈圖
X 軸＝操縱變因，Y 軸＝應變變因
加誤差棒（±uc）
用圖形論證你的結論
💡 「沒有顯著差異」也是重要的結論——但你需要用不確定度和圖形說明「為什麼這個差異不算有意義」。
📈 數據會說話——但你要會聽
收完數據別急著寫結論，先問自己三個問題：
① 趨勢：散佈圖的點大致往哪走？上升？下降？先升後降？沒有趨勢？
② 離散：點是緊貼趨勢線，還是散得很開？散開代表不確定度大，結論要保守。
③ 意外：有沒有哪個點特別「跳出來」？它可能是操作失誤，也可能是最有趣的發現。
能回答這三題，你就有資格寫結論了。不確定怎麼畫圖？→ Excel 工具箱
✏️ 用 2–3 句話寫下你的實驗結論（包含結果、證據、與預測的比較）

🤖 AI 檢核站

完成你的實驗計畫後，把計畫交給 AI（ChatGPT、Perplexity 等），請它幫你找漏洞：

「我的控制變因有沒有遺漏？」
「這個實驗設計有什麼潛在問題？」
「你覺得我的數據量夠嗎？需要幾組重複？」
💬 複製貼上範本：「我正在設計一個實驗，研究問題是＿＿。操縱變因是＿＿（水準：＿＿），應變變因是＿＿，控制變因有＿＿。請幫我檢查：(1) 控制變因有沒有遺漏 (2) 實驗步驟有沒有潛在問題 (3) 數據量夠不夠」
給 AI 越多上下文，它的回饋越有用。空白處填入你自己的內容再送出。

⚠️ AI 是審稿人，不是代筆人。它指出問題，你決定要不要改、怎麼改。

📂 學習歷程提示

「獨立選定變因並設計驗證」和「用不確定度解讀結果」最有價值。實驗計畫→數據表→散佈圖→結論，完整呈現你的科學思維。同儕互評表也是好素材。

✍️ 反思錨點（寫 2-3 句就好）：
你原本的實驗計畫和實際做的有什麼不同？改了什麼？為什麼改？
「計畫 vs 實際」的落差本身就是最好的反思素材——改得好代表你在成長。

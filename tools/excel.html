📊 Excel 數據分析指南

實驗數據整理的基本技能——從輸入到圖表的完整流程

Lv2 W5–W9 常用
一、建立數據表的規範
❌ 常見錯誤
  • 欄位沒有標題（不知道是什麼數據）
  • 沒有單位（「1.42」不知道是秒還是公分）
  • 手算四捨五入後直接填入（失去精度）
  • 把數字和文字混在同一格（公式會失效）
✅ 正確做法
  • 第一列：欄位標題（如：質量 m (g)）
  • 單位寫在標題括號裡，數字裡不要帶單位
  • 保留原始數字，用公式欄位另外算平均
  • 每格只填一個數值（不要「1.42 ± 0.03」放同格）
二、常用函數速查
函數用途範例說明
=AVERAGE(A2:A6) 計算平均值 A 欄 5 筆數據的平均 最常用，取代手算
=MAX(A2:A6)-MIN(A2:A6) 計算極差 最大值 − 最小值 估不確定度的簡單方法
=STDEV(A2:A6) 樣本標準差 A 欄 5 筆數據的標準差 比極差更準確的離散程度
=LOG10(A2) 常用對數（log₁₀） log₁₀(A2) 的值 線性化冪次關係用
=LN(A2) 自然對數（ln） lnA2 的值 線性化指數關係用
=A2^2 平方 A2 的平方 T² vs L 線性化時用
=SQRT(A2) 開根號 √A2 計算不確定度傳遞時用
三、畫散佈圖（XY 圖）完整步驟
1
選取數據範圍
先點 X 欄標題，按住 Ctrl 再點 Y 欄標題（不含單位行，只選數字區域）。
⚠️ 不要選到文字欄！Excel 會把文字當 0 處理，圖形會出錯。
2
插入 → 圖表 → XY 散佈圖
選「散佈圖（只有標記，沒有連線）」——不要選折線圖！折線圖會依照順序連點，不是按 X 值排列。
3
加趨勢線
右鍵點圖上的數據點 → 「加趨勢線」→ 選「線性」（通常）→ 勾選：
☑ 在圖表上顯示方程式
☑ 在圖表上顯示 R² 值
4
編輯軸標題和單位
雙擊 X 軸 / Y 軸 → 在「格式化座標軸」中修改。或點圖表右上角的「+」加入「軸標題」。
格式：物理量名稱（符號）/ 單位，例如：質量 m / g 或 週期 T / s
✓
完成檢查
最終圖表應包含：標題、X 軸標籤（含單位）、Y 軸標籤（含單位）、趨勢線、方程式、R² 值。
四、R² 是什麼？怎麼解讀？

R²（決定係數）告訴你：數據點和趨勢線符合的程度。

0.99+
優秀。數據點幾乎都在趨勢線上。
0.90–0.98
良好。略有離散，但關係清楚。
< 0.90
需要注意。數據很分散，趨勢線不可靠。
💡 R² 低不一定代表實驗失敗——可能是關係根本不是線性的（例如冪次或指數），試試看 log-log 圖或半 log 圖。
五、log-log 圖（冪次關係）的操作
當你懷疑兩個量是冪次關係（y = a·xⁿ）
步驟 1：新增 log 欄位
在 C 欄填入：=LOG10(A2)　（對 X 值取 log）
在 D 欄填入：=LOG10(B2)　（對 Y 值取 log）
步驟 2：用 C、D 欄畫散佈圖
這樣畫出來的是 log Y vs log X，如果是冪次關係，圖形會是直線。
步驟 3：讀取斜率
趨勢線的斜率 = 冪次 n。截距 = log a，所以 a = 10^(截距)。
